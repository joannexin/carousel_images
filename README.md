# carousel_images

It has been built by Javascript and node.js


## Install

Step 1: Download/clone.
```
git clone https://github.com/joannexin/carousel_images.git
```
Step 2: Install dependencies.
```
npm install
```
Step 3: Start server.
```
node server.js
```
Step 4: Open http://localhost:3000/ in your browser.
